#### Usage:
1. Get code:
   - Clone 
    ```
    git clone https://github.com/lacee677/gomba_programscreen_setup.git
    ```
    
   - Wget  
    ```
    wget https://raw.githubusercontent.com/lacee677/gomba_programscreen_setup/master/gomba_teszt.sh
    ```
    
2. `sudo chmod u+x gomba_teszt.sh`
3. `./gomba_teszt.sh`
